Rice distribution proof

Published

October 11, 2024

정규분포와 Rice 분포의 상관관계

Rice 분포는 두 개의 독립적인 정규분포(가우시안 분포)를 가진 확률 변수의 크기(벡터의 길이)에 대한 분포이다. 따라서 Rice 분포는 정규분포와 밀접한 관계를 가지고 있다.

1. 관계의 기본 개념

  • 정규분포: 평균 \(\mu\)와 분산 \(\sigma^2\)를 가지는 연속 확률 분포로, 실수 값을 가지는 하나의 확률 변수를 모델링합니다.
  • Rice 분포: 두 개의 독립적인 정규분포 확률 변수 \(X\)와 \(Y\)의 크기 \(R = \sqrt{X^2 + Y^2}\)의 분포이다.

2. 수학적 유도

두 개의 독립적인 정규분포 확률 변수 \(X\)와 \(Y\)는 다음과 같은 확률분포를 따른다:

  • \(X \sim N(\mu_x, \sigma^2)\)
  • \(Y \sim N(\mu_y, \sigma^2)\)

이때, 확률 변수 \(R\)는 다음과 같이 정의 된다:

\[ R = \sqrt{X^2 + Y^2} \]

\(R\)은 Rice 분포를 따르며, Rice 분포의 파라미터는 다음과 같이 설정 된다:

  • 비중심성 파라미터 \(\nu = \sqrt{\mu_x^2 + \mu_y^2}\)
  • 스케일 파라미터 \(\sigma\)

3. Rice 분포의 확률 밀도 함수(PDF)

Rice 분포의 확률 밀도 함수는 다음과 같다:

\[ f_R(r) = \frac{r}{\sigma^2} \exp\left( -\frac{r^2 + \nu^2}{2\sigma^2} \right) I_0\left( \frac{r \nu}{\sigma^2} \right), \quad r \geq 0 \]

  • \(I_0\)는 0차 수정 베셀 함수이다.

4. 베셀 함수가 Rice 분포의 PDF에 등장하는 이유

베셀 함수는 Rice 분포의 확률 밀도 함수를 유도하는 과정에서 각도에 대한 적분을 할 때 자연스럽게 등장한다. 이는 두 개의 정규분포 확률 변수의 크기를 구하는 과정에서 발생한다.

4.1 문제 설정

두 개의 독립적인 정규분포 확률 변수 \(X\)와 \(Y\):

  • \(X \sim N(\mu_x, \sigma^2)\)
  • \(Y \sim N(\mu_y, \sigma^2)\)

벡터의 크기 \(R\):

\[ R = \sqrt{X^2 + Y^2} \]

목표: \(R\)의 확률 밀도 함수 \(f_R(r)\)를 구한다.

4.2 결합 확률 밀도 함수와 극좌표 변환

결합 확률 밀도 함수(Joint probability distribution):

\[ f_{X,Y}(x, y) = \frac{1}{2\pi \sigma^2} \exp\left( -\frac{(x - \mu_x)^2 + (y - \mu_y)^2}{2\sigma^2} \right) \]

극좌표 변환을 적용한다:

  • \(x = r \cos \theta\)
  • \(y = r \sin \theta\)
  • 야코비안(Jacobian) \(J = r\)

4.3 \(R\)의 확률 밀도 함수 유도

\(R\)의 확률 밀도 함수는 다음과 같다:

\[ f_R(r) = \int_{0}^{2\pi} f_{X,Y}(r \cos \theta, r \sin \theta) \cdot r \, d\theta \]

이를 대입하면:

\[ f_R(r) = \frac{r}{2\pi \sigma^2} \int_{0}^{2\pi} \exp\left( -\frac{(r \cos \theta - \mu_x)^2 + (r \sin \theta - \mu_y)^2}{2\sigma^2} \right) d\theta \]

4.4 지수 함수 내의 식 전개 및 단순화

지수 함수 내의 식을 전개하면 다음과 같이 나타난다:

\[ \begin{align*} (r \cos \theta - \mu_x)^2 + (r \sin \theta - \mu_y)^2 &= r^2 - 2 r \mu_x \cos \theta + \mu_x^2 + r^2 - 2 r \mu_y \sin \theta + \mu_y^2 \\ &= 2 r^2 - 2 r (\mu_x \cos \theta + \mu_y \sin \theta) + (\mu_x^2 + \mu_y^2) \end{align*} \]

이를 단순화하면:

\[ (r \cos \theta - \mu_x)^2 + (r \sin \theta - \mu_y)^2 = r^2 + \nu^2 - 2 r \nu \cos(\theta - \phi) \]

여기서:

  • \(\nu = \sqrt{\mu_x^2 + \mu_y^2}\)
  • \(\tan \phi = \frac{\mu_y}{\mu_x}\)
  • \(\mu_x = \nu \cos \phi\)
  • \(\mu_y = \nu \sin \phi\)

지수 함수 내의 전체 식을 전개하면:

\[ -\frac{(r^2 + \nu^2 - 2 r \nu \cos(\theta - \phi))}{2\sigma^2} = -\frac{r^2 + \nu^2}{2\sigma^2} + \frac{r \nu}{\sigma^2} \cos(\theta - \phi) \]

4.5 적분 부분에서 베셀 함수 등장

따라서 \(f_R(r)\)는 다음과 같이 표현 된다:

\[ f_R(r) = \frac{r}{2\pi \sigma^2} \exp\left( -\frac{r^2 + \nu^2}{2\sigma^2} \right) \int_{0}^{2\pi} \exp\left( \frac{r \nu}{\sigma^2} \cos(\theta - \phi) \right) d\theta \]

여기서 적분 부분은 베셀 함수로 나타 낼 수 있다:

\[ \int_{0}^{2\pi} \exp\left( \kappa \cos \theta \right) d\theta = 2\pi I_0(\kappa) \]

  • \(\kappa = \frac{r \nu}{\sigma^2}\)
  • \(I_0(\kappa)\)는 0차 수정 베셀 함수이다.

4.6 최종 확률 밀도 함수

적분 결과를 대입하면:

\[ f_R(r) = \frac{r}{\sigma^2} \exp\left( -\frac{r^2 + \nu^2}{2\sigma^2} \right) I_0\left( \frac{r \nu}{\sigma^2} \right) \]

따라서 Rice 분포의 확률 밀도 함수에 베셀 함수 \(I_0\)가 등장하게 된다.

5. 결론

  • 정규분포와 Rice 분포의 관계: 두 개의 독립적인 정규분포 확률 변수의 크기는 Rice 분포를 따른다.
  • 베셀 함수의 등장 이유: 극좌표 변환 후 각도에 대한 적분을 수행하는 과정에서 베셀 함수가 자연스럽게 나타난다.
  • 의의: 이러한 유도 과정은 신호 처리, 통신 이론 등에서 중요한 역할을 한다.
